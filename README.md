An award-winning style, interactive 3D web experience that takes users on a cinematic descent into an oceanic trench to discover lost relics. The journey seamlessly transitions into an interactive, isometric 3D museum gallery.

---

## ✨ Features

* **Cinematic Scroll Experience:** A WebGL-powered deep-sea descent synchronized with GSAP scroll triggers.
* **Photorealistic Rendering:** Utilizes `ACESFilmicToneMapping` and `SRGBColorSpace` alongside custom directional lighting, fill lights, and animated God Rays for a premium visual aesthetic.
* **Interactive Isometric Gallery:** A standalone 3D archive room where users can scroll to inspect individual 3D artifacts resting on a central focal table, complete with holographic UI data overlays.
* **Immersive UI/UX:** Custom loading screens, depth-tracking HUD, spatial audio toggles, and seamless GSAP camera transitions.
* **Optimized Performance:** Built with Angular standalone components and efficient Three.js resource management (geometry disposal, loading managers).

---

## 🛠️ Tech Stack

* **Framework:** Angular (Standalone Components)
* **3D Engine:** Three.js
* **Animations:** GSAP (GreenSock Animation Platform) + ScrollTrigger
* **Post-Processing:** UnrealBloomPass (Three.js examples)
* **Styling:** SCSS
